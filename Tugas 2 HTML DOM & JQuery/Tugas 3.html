FAQ Interaktif

Apa itu jQuery? ▶
Pustaka JavaScript yang mempercepat dan menyederhanakan pengembangan web, khususnya untuk manipulasi HTML, penanganan event, animasi, dan AJAX dengan sintaksis yang ringkas.
Apakah jQuery bahasa pemrograman? ▶
Bukan, jQuery adalah pustaka (library) JavaScript, bukan bahasa pemrograman terpisah. Ini dibangun di atas JavaScript.
Bagaimana cara menggunakan jQuery? ▶
Dengan menyertakan file jquery.js (atau via CDN) di halaman HTML Anda, lalu Anda bisa langsung menggunakan fungsi-fungsinya seperti $(selector).action().
Apa perbedaan antara .hover() dan CSS :hover? ▶
CSS :hover murni untuk tujuan gaya visual sederhana. .hover() jQuery adalah fungsi pintasan untuk mengikat dua penangan acara: satu untuk mouseenter (saat masuk) dan satu lagi untuk mouseleave (saat keluar), memungkinkan Anda menjalankan logika JavaScript yang lebih kompleks (seperti menambahkan/menghapus kelas dinamis, bukan hanya mengubah warna) saat kursor masuk atau keluar.
Metode .animate() digunakan untuk apa? ▶
Membuat efek animasi kustom dengan mengubah properti CSS numerik secara bertahap (misalnya, opacity, width).
